Pandas CSV Reader & Basic Analysis

This project demonstrates basic data analysis using Pandas.

Features

- Read CSV dataset
- Display first 5 rows (head)
- Display last 5 rows (tail)
- Check data types
- Calculate mean, median, minimum and maximum values
- Filter rows based on conditions
- Select specific columns
- Export results to a new CSV file

Tools Used

- Python
- Pandas
- Google Colab

- Untitled6.ipynb
